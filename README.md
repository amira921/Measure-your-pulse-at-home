<h1>Measure your Pulse at Home</h1>

<h2>The Goal of the Project </H2>
Heartbeat Sensor is an electronic device that is used to measure the heart rate. </br> </br>

<H2>Components required</H2>

- Arduino UNO  
- Arduino UNO Cable 
- LCD
- Jumper Wire (M to M)  
- 10k Potentiometer 
- Breadboard  
- Heart pulse sensor
